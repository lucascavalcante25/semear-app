# Respostas para Objeções — Prospecção de Igrejas

> Estrutura de cada resposta: validar a preocupação → mostrar que entendeu →
> conectar a um benefício real → pergunta calibrada. Sem confronto, sem promessas
> não confirmadas. Três versões por objeção: **curta**, **consultiva** e **para áudio**.
>
> Valores reais: mensal R$ 57,00 · anual PIX R$ 570,00 · adesão única R$ 200,00 ·
> teste grátis 7 dias · sem limite de membros no lançamento.

---

## 1. "Não temos interesse."
**Curta:**
```
Entendo, pastor. Respeito totalmente. Se um dia quiserem reunir as informações da
igreja em um lugar só, fico à disposição. Posso deixar meu contato salvo com o senhor?
```
**Consultiva:**
```
Tudo bem, pastor, agradeço a sinceridade. Só para eu entender melhor e não insistir
à toa: o que hoje já funciona bem na organização da igreja?
```
**Áudio:**
```
Pastor, sem problema nenhum, tá? Respeito bastante. Fica meu contato salvo aí, e se
um dia fizer sentido reunir tudo num lugar só, é só me chamar. Um abraço.
```

## 2. "Não precisamos disso agora."
**Curta:**

```
**Consultiva:**
```
Entendo, pastor. Imagino que agora o foco esteja em outras prioridades. O que estaria
à frente disso na igreja neste momento?
```
**Áudio:**
```
Tranquilo, pastor. Nada urgente. Deixa eu te mandar um resumo pra ficar guardado, e
quando for um bom momento a gente conversa com calma. Combinado?
```

## 3. "Já usamos planilha."
**Curta:**
```
Ótimo, pastor. Planilha organiza bastante coisa mesmo. O sistema entra para reunir
tudo em um lugar só e acessar pelo celular. Como vocês organizam essas planilhas hoje?
```
**Consultiva:**
```
Entendo, e planilha ajuda muito no começo. Pelo que percebo, o desafio costuma ser
quando a informação cresce e fica em vários arquivos. O que dá mais trabalho hoje
para manter tudo atualizado?
```
**Áudio:**
```
Perfeito, pastor. Planilha resolve bastante coisa. A ideia do sistema é justamente
juntar tudo num lugar só e deixar acessível no celular. Me conta: como vocês mexem
nessas planilhas hoje?
```

## 4. "Nossa igreja é pequena."
**Curta:**
```
Entendo, pastor. E funciona bem também para igrejas menores — sem limite de membros
no lançamento e com a equipe aprendendo rápido. O que hoje dá mais trabalho de
organizar aí?
```
**Consultiva:**
```
Faz sentido, pastor. Igreja menor costuma ter equipe enxuta, e é aí que reunir tudo
num lugar só ajuda a não sobrecarregar ninguém. Quantas pessoas cuidam da parte
administrativa hoje?
```
**Áudio:**
```
Pastor, sem problema, o sistema atende bem igreja menor também. Aliás, no lançamento
não tem limite de membros. Me diz uma coisa: o que hoje dá mais trabalhinho pra
organizar aí na igreja?
```

## 5. "Não temos orçamento."
**Curta:**
```
Entendo, pastor. Orçamento da igreja é para ser cuidado mesmo. O teste é grátis por 7
dias, sem cartão, então dá para conhecer sem custo. Faria sentido começar por aí?
```
**Consultiva:**
```
Compreendo totalmente. Pelo que entendi, o momento pede cautela com gastos. Se, ao
testar, a igreja vir que economiza tempo da equipe, o valor de R$ 57 por mês faria
sentido para vocês?
```
**Áudio:**
```
Pastor, entendo bem, orçamento é pra cuidar mesmo. Olha, o teste é grátis por 7 dias,
sem cartão. Dá pra conhecer sem gastar nada. Quer começar por aí, só pra ver como é?
```

## 6. "Está caro."
**Curta:**
```

```
**Consultiva:**
```
Respeito a sua avaliação, pastor. Talvez o valor pareça alto sem ver o que ele
substitui em tempo e organização. O que hoje mais consome tempo da equipe?
```
**Áudio:**
```
Entendo, pastor. Deixa eu te explicar melhor o que tá incluído, porque o mensal é R$
57 e o anual ainda dá 2 meses grátis. Me diz: o que pra você faria isso valer a pena?
```

**Curta:**
```

```

```
Pastor, entendo, tempo é curto pra todo mundo. Por isso o sistema foi feito pra ser
simples, e a gente dá uma orientação no começo. Quem seria a pessoa que cuidaria disso
aí na igreja?
```

## 12. "Quem vai alimentar o sistema?"
**Curta:**
```
Ótima pergunta, pastor. Normalmente a secretaria ou um líder cuida disso, e cada
pessoa acessa conforme o papel. Quem hoje já cuida dessas informações aí?
```
**Consultiva:**
```
Entendo a preocupação. Pelo que percebo, o receio é gerar mais trabalho. A ideia é o
contrário: cada função tem sua área e o registro fica organizado. Como isso é dividido
hoje na equipe?
```
**Áudio:**
```
Boa pergunta, pastor. Em geral é a secretaria ou um líder que alimenta, e cada um
acessa a parte dele. Me conta: quem hoje já cuida dessas informações aí na igreja?
```

## 13. "Temos receio de colocar os dados dos membros."
**Curta:**
```
Compreendo totalmente, pastor. Cuidar dos dados das pessoas é sério mesmo. O acesso é
por login e por perfil, então cada um vê só o que lhe cabe. Que tipo de dado mais
preocupa vocês?
```
**Consultiva:**
```
Faz todo sentido essa preocupação. Pelo que entendi, o cuidado é com a privacidade dos
membros. O sistema trabalha com acesso por perfil e login. O que a igreja consideraria
essencial para se sentir segura nesse ponto?
```
**Áudio:**
```
Pastor, entendo perfeitamente, dado de membro é coisa séria. O acesso é por login e
por perfil, então cada pessoa vê só o que é da função dela. Me diz: que tipo de
informação mais preocupa vocês nesse ponto?
```
> Nota interna: não afirmar conformidade jurídica/LGPD sem confirmação do proprietário.

## 17. "Quanto custa?"
**Curta:**
```
Com prazer, pastor: mensal de R$ 57, anual à vista de R$ 570 no PIX (2 meses grátis) e
adesão única de R$ 200 na ativação, com 7 dias de teste grátis. Quer que eu mostre o
que está incluído?
```
**Consultiva:**
```
Claro. Antes do número, vale ver o que ele substitui em tempo e organização. Em resumo:
R$ 57/mês ou R$ 570/ano no PIX, adesão única de R$ 200 e 7 dias grátis. Posso te mostrar
na prática?
```
**Áudio:**
```
Claro, pastor. É R$ 57 por mês, ou R$ 570 no ano à vista no PIX, que dá 2 meses grátis.
Tem uma adesão única de R$ 200 na ativação e 7 dias de teste grátis. Quer que eu te
mostre o que vem incluído?
```

**Consultiva:**
```
Sim. Pelo que percebo, é importante ter com quem contar depois. O suporte é feito pela
central dentro do sistema. O que seria essencial para vocês em termos de atendimento?

**Áudio:**
```

relatar problema ou mandar sugestão, e a equipe acompanha. Isso te ajuda a ficar mais
tranquilo?
```
> Nota interna: não prometer "suporte 24h" nem SLA. Horário/canal exato **pendente de confirmação**.
