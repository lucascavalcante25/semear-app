# Follow-up Comercial — Acompanhamento Respeitoso

> Objetivo: manter o contato vivo sem pressionar. Frases leves, autonomia para o
> pastor, sempre preservando a possibilidade de retorno futuro.
>
> **Nunca usar:** "Você não respondeu", "Estou aguardando", "Já mandei várias
> mensagens", "Não vai me dar um retorno?".
>
> **Regra de ouro:** no máximo **3 tentativas seguidas** sem resposta. Depois disso,
> pausar e só retomar após 30 dias (ou quando a igreja pedir).

---

## Intervalos sugeridos entre contatos

| Etapa | Quando enviar |
|-------|----------------|
| 1º follow-up | 2 a 3 dias após a mensagem sem resposta |
| 2º follow-up | 4 a 5 dias após o 1º follow-up |
| Último contato | 7 dias após o 2º follow-up |
| Após enviar demonstração | 2 dias depois |
| Após reunião | 1 dia depois (no mesmo dia ou no dia seguinte) |
| Após início do teste | No 3º dia e no 6º dia do teste (teste dura 7 dias) |
| "Mais para frente" | Retomar em 30 dias |
| "Falar com a diretoria" | 5 a 7 dias depois |
| Retomada geral | 30 dias após o último contato |

---

## 1. Primeiro follow-up (sem resposta)
```
Pastor, tudo bem? Passando só para saber se o senhor chegou a ver a mensagem que
enviei. Se este não for um bom momento, fique totalmente à vontade para me avisar.
```

## 2. Segundo follow-up
```
Pastor, tudo certo? Deixo o convite de pé, sem pressa. Se quiser, posso enviar uma
demonstração rápida para o senhor conhecer quando tiver um tempinho.
```

## 3. Último contato (da sequência)
```
Pastor, imagino que a rotina esteja cheia. Vou deixar meu contato salvo com o senhor
e paro por aqui para não incomodar. Quando fizer sentido para a igreja, é só me
chamar. Fico à disposição, tá?
```

## 4. Contato depois de enviar a demonstração
```
Pastor, passando apenas para confirmar se conseguiu visualizar o material. Caso este
não seja um bom momento, fique à vontade para me avisar. O que chamou mais a atenção?
```

## 5. Contato depois da reunião
```
Pastor, obrigado novamente pela conversa de ontem. Fiquei à disposição para qualquer
dúvida que tenha surgido. O senhor gostaria de dar o próximo passo pelo teste de 7
dias?
```

## 6. Contato depois do início do teste
```
Pastor, tudo bem? Como está sendo a experiência com o sistema até aqui? Se aparecer
qualquer dúvida, estou por perto para ajudar. Tem algum módulo que o senhor queira
que eu mostre melhor?
```

## 7. Contato quando a igreja disser "mais para frente"
```
Pastor, tudo bem? Como o senhor comentou que faria mais sentido em outro momento,
passei só para deixar o convite de pé. Seguimos no ritmo da igreja. Este período está
melhor para conversarmos, ou prefere que eu retorne mais adiante?
```

## 8. Contato quando precisar falar com a diretoria
```
Pastor, tudo certo? Passando para saber se o senhor precisa de algum material extra
para apresentar à diretoria. Posso preparar um resumo com os valores e os módulos, se
ajudar. Como costuma ser o tempo dessas conversas aí na igreja?
```

## 9. Retomada após 30 dias
```
Pastor, a paz! Faz um tempo que conversamos sobre o Minha Igreja Digital. Passando
para saber como está a organização da igreja neste período e se faria sentido
retomarmos aquela ideia. Sem compromisso, claro.
```

## 10. Encerramento educado
```
Pastor, agradeço demais a atenção que o senhor teve comigo. Vou deixar meu contato
salvo e encerro por aqui para respeitar o momento da igreja. Se um dia quiserem
conhecer, será um prazer ajudar. Que Deus abençoe o senhor e a igreja!
```

---

## Boas práticas de follow-up

- Sempre dar uma saída fácil ao pastor ("fique à vontade para avisar").
- Cada follow-up deve trazer algo novo (uma pergunta, um material, uma opção) — nunca
  só "e aí?".
- Terminar com pergunta simples quando fizer sentido.
- Se o pastor pedir para parar, respeitar imediatamente e agradecer.
- Registrar cada contato no funil (ver `docs/guia-prospeccao-igrejas.md`).
